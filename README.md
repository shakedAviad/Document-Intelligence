# Document Agent — Starter Repository

This is the starter repository for the Document Agent home assignment.

## Contents

The `documents/` folder contains the following files your agent should be able to work with:

| File | Format | Description |
|------|--------|-------------|
| `meetings.md` | Markdown | Meeting notes from a project team (3 meetings) |
| `sales-q1.csv` | CSV | Q1 sales data — note: contains some inconsistencies |
| `emails.txt` | Plain text | Email thread between team members about revenue discrepancies |
| `config.json` | JSON | Production application configuration |
| `server-log.txt` | Plain text | Server log with timestamps, errors, and warnings |

## Getting Started

1. Clone this repository
2. Build your Document Agent (see assignment document for full details)
3. Update this README with instructions on how to run your solution
